## Project Overview

Rafiki Guidance Hub is a comprehensive student support platform with the following key features:

### Core Features

- **AI-Powered Chat Interface**: Natural conversation with contextual understanding of student needs
- **Career Guidance**: Assessments, recommendations, and resources for career planning
- **Wellbeing Support**: Mental health assessments, resources, and support tools
- **Assessment System**: Various assessments for career, wellbeing, and academic purposes
- **Personalized Dashboard**: Customized overview of activities, recommendations, and resources

### Technology Stack

- **Frontend**: React, TypeScript, Vite
- **Styling**: TailwindCSS, shadcn/ui
- **Authentication**: Supabase Auth
- **Database**: Supabase PostgreSQL
- **AI Integration**: Custom AI models via Supabase Edge Functions
